# Belly Fat Sleep Tracking Workflow

## Article Summary: "Your Belly Fat Isn't From Food"

**Core Thesis**: Belly fat accumulation is primarily driven by sleep disruption, not caloric intake. The critical window is **10pm–2am** for deep sleep and growth hormone release.

**Key Mechanism**: Growth hormone peaks in the first 90 minutes of deep sleep. Missing this peak causes the body to store tomorrow's calories as visceral fat instead of repairing muscle.

## Assistant App Tracking Workflow

### 1. Daily Tracking Dashboard

#### Sleep Metrics
| Metric | Target | Tracking Method |
|--------|--------|-----------------|
| Lights out | ≤10:00pm | Manual entry / bedtime reminder log |
| Deep sleep window (10pm-2am) | 4+ hours | Sleep tracking integration (optional) |
| Phone out of bedroom | By 10:30pm | Yes/No toggle |
| Bedroom temperature | 65-68°F (18-20°C) | Manual entry |

#### 3-Step Wind-Down Protocol
| Step | Time | Action | Completion |
|------|------|--------|------------|
| Overhead lights off | 90 min before bed | Switch to lamp lighting | ✓ / ✗ |
| Screen cutoff | 45 min before bed | Book/blue-light glasses only | ✓ / ✗ |
| Cool + empty + out | 15 min before bed | 65-68°F, priorities written, phone in another room | ✓ / ✗ |

#### Food Tracking
| Category | Guidelines | Tracking |
|----------|------------|----------|
| Dinner timing | Before 8pm | Time logged |
| Protein | Palm-sized portion | ✓ / ✗ |
| Slow carb | Fist-sized portion | ✓ / ✗ |
| Alcohol | Avoid after 8pm | ✓ / ✗ |
| Caffeine | Cutoff 2pm | ✓ / ✗ |
| Kiwi/magnesium | 8pm if needed | ✓ / ✗ |

#### Activity Swap (Short Sleep Days)
| Sleep <6 hours | Do Instead of Normal Workout |
|----------------|------------------------------|
| ✓ | 20-min walk OR light mobility circuit |
| ✗ | Skip heavy lifts/HIIT |

### 2. Weekly Progress Review

#### Week Structure (7-Day Reset)
- **Days 1-2**: Basic sleep window + simple dinner protocol
- **Days 3-4**: Add full wind-down protocol
- **Days 5-7**: Full food rules implementation

#### Weekly Metrics to Track
1. **Adherence Score**: % of days meeting all 3 wind-down steps
2. **Belly Fat Proxy**: Waist measurement (morning, before eating)
3. **Energy Levels**: 1-10 scale (morning and afternoon)
4. **Cravings**: Track late-night food desires
5. **Sleep Quality**: Subjective 1-10 scale

### 4. Decision Trees for Edge Cases

#### Shift Work / Irregular Schedule
```
IF work schedule shifts bedtime past 10pm:
  → Adjust wind-down to 90/45/15 min before ACTUAL sleep
  → Still target 4+ hours sleep before midnight when possible
```

#### Partner/Wife TV in Bed
```
IF partner watches TV:
  → Use blue-light glasses + book
  → Earplugs / white noise for your sleep
  → Negotiate TV curfew (e.g., 10:30pm)
```

#### Bad Back / Injury
```
IF injured:
  → Replace workout swap with:
    - Gentle stretching (10 min)
    - Mobility focus on non-injured areas
    - Pool/water therapy if available
```

#### Baby Waking at 3am
```
IF parenting interruptions:
  → Focus on protecting the 10pm-2am window when possible
  → Quick nap (20 min) if nighttime wake-ups occur
  → Morning recovery meal still critical
```

### 5. Measurement Protocol

#### Weekly Check-ins (Same Day, Same Time)
- **Waist measurement**: At navel level, relaxed breath
- **Weight**: Same scale, same time of day
- **Subjective scores**: Energy, mood, cravings (1-10)

#### Monthly Assessments
- **Progress photos**: Front/side/back
- **Blood work** (optional): If available, track inflammatory markers
